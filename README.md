## 如何运行  

**添加vscode中python插件**  

**创建一个虚拟环境**  
```
py -3 -m venv .venv
.venv\scripts\activate
```

**安装packages**  
```
python -m pip install nltk
python -m pip install xlwt
```

**首次运行**  
运行`nltk.download(...)`，之后运行即可注释掉

**运行参数**  
输入正确函数参数
